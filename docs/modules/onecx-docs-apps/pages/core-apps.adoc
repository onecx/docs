= OneCX Core Applications

:idprefix:
:idseparator: -
:imagesdir: ../images

A Core Application is an essential part of the OneCX platform that provides fundamental functionalities required for the system to operate effectively.

== Overview

[.stripes-even,cols="30,~"]
|===
| Application | Description

| xref:documentation:onecx-data-orchestrator:page$index.adoc[Data Orchestrator]
| Manages data synchronization and integration between different OneCX services and external systems.

| xref:documentation:onecx-iam:page$index.adoc[Identity Access Management]
| Handles user authentication, authorization, and identity management within the OneCX platform.

| xref:documentation:onecx-parameter:page$index.adoc[Parameter Management]
| Used by some applications

| xref:documentation:onecx-permission:page$index.adoc[Permission Management]
| Declared and used in applications.

| xref:documentation:onecx-product-store:page$index.adoc[Application Store]
| Application data (components, slots etc.)

| xref:documentation:onecx-shell:page$index.adoc[Shell]
| The main user interface framework for OneCX applications.

| xref:documentation:onecx-tenant:page$index.adoc[Tenant Management]
| Represents a customer or organizational unit using OneCX

| xref:documentation:onecx-theme:page$index.adoc[Theme Management]
| Customizing the look and feel of OneCX for a tenant

| xref:documentation:onecx-theme:page$index.adoc[User Profile]
| Managing user-specific settings and preferences

| xref:documentation:onecx-workspace:page$index.adoc[Workspace Management]
| Customizing the working areas for users
|===
