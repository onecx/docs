= OneCX Feature Applications

:idprefix:
:idseparator: -
:imagesdir: ../images

A Feature Application is an optional part of the OneCX platform that provides additional functionalities to enhance the system's capabilities.


.OneCX Feature Applications

|===
| Application | Description

| xref:documentation:onecx-announcement:page$index.adoc[Announcement Management]

| xref:documentation:onecx-documentation:page$index.adoc[Documentation Management]
| Managing documents in a user-friendly and efficient manner

| xref:documentation:onecx-help:page$index.adoc[Help Management]

| xref:documentation:onecx-welcome:page$index.adoc[Welcome Management]
