= Get OneCX

Welcome to OneCX. +
This guide gives you a clear, reliable path how to get started with OneCX. +
It is grounded in the xref:onecx-local-env:index.adoc[OneCX Local Environment] - start with this.

After initial setup, the local environment provides the core infrastructure and applications needed to start developing microfrontends and services within OneCX. +
Also you can explore building microfrontends, services, and extensions using the xref:onecx-docs-dev:index.adoc[OneCX Developer Documentation].

Finally, you can leverage the xref:onecx-nx-plugins:index.adoc[OneCX App Generator] to streamline your development workflow with powerful generators and tools tailored for the OneCX framework.
